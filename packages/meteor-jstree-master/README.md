# meteor-jstree

[jsTree](http://www.jstree.com/) is jQuery plugin, that provides interactive trees. It's very popular plugin with a lot of features. But [jsTree](http://www.jstree.com/) did not have a native support of Meteor Framework.

Main purpose of `jss:jstree` package â€” make [jsTree](http://www.jstree.com/) more reactive and Meteor-compatible. Using this package you can build tree from reactive data source (Collection, ReactiveVar, Session, ReactiveObject, etc).
